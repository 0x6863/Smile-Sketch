## 功能列表

### 一键导出

> 按下快捷键 `ctrl shift e` 后，插件自动扫描全部页面，直接导出全部可用资源到设计文件所在目录下

**自定义导出目录**

如：

- Test.sketch
  - Page 1
  - Page 2
  - Page 3

自定义路径方法：在页面名称后添加 ` | Path` 来实现路径修改

Path 的几种写法

- Page 1
  - 导出在设计稿同目录下
- Page 1 | 导出
  - 导出在设计稿同目录下的导出目录中
- Page 1 | ..
  - 导出在设计稿的上级目录下
- Page 1 | ~/Desktop
  - 导出在当前用户的桌面下
